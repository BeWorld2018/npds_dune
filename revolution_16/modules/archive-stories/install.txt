----------------------------------------------------------------------
Installation : archive-stories
----------------------------------------------------------------------
 - Créer un sous répertoire [archive-stories] dans le répertoire modules
----------------------------------------------------------------------

Vous pouvez configurer le module
 - via cette url :
   admin.php?op=Extend-Admin-SubModule&ModPath=archive-stories&ModStart=admin/archive-stories_set
----------------------------------------------------------------------
Lancement : archive-stories
----------------------------------------------------------------------
 - via une url de type :
   modules.php?ModPath=archive-stories&ModStart=archive-stories
