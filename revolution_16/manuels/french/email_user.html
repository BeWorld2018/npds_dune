Diffusion d'un Message Interne


L'objectif de cette fonction est de vous permettre de communiquer rapidement avec les membres de votre site.
Cette fonction ne doit pas être utilisée comme système de NewsLetter (NPDS dispose pour cela d'une application beaucoup plus puissante : LNL).


Vous pouvez envoyer un Message Interne à un utilisateur précis (saisir son Surnom), à un groupe de membres, à tous les membres du portail.

Saisissez le sujet et le texte. Pour finir cliquez sur le bouton OK.
