Ce thème se veut Modulable à souhait sans grandes connaissances mais demande malgrés tout de la rigueur dans ses modifications. Il utilise comme base la feuille de style proposée par Alsacréation: Knacss

  • Vous trouverez le code commenté de cette feuille de style de base ICI
  • Il est donc sur une structure dite "fluide" qui permettra un redimentionnement du site en fonction de votre résolution d'écran
  • Vous trouvez aussi une page "Central.php" à la racine du theme et une page "Central.html" qui vous permettra de personnailer, si besoin, votre page d'accueil, et ceci trés simplement; je vous renvoi vers le post de Jireck sur le sujet qui se trouve ICI
  • Le menu responsive est basique,

Ultime question: puis je modifier ceci depuis l'admin de Npds online? Réponse OUI grace au Gestionnaire de fichier qui intègre l'éditeur TinyMce.
