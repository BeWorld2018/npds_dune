Bienvenue dans les pages d'administration de NPDS.

NPDS est un générateur de portail Français en Open-Source sous licence GNU-GPL (Gratuit) utilisant les technologies PHP et MySql.

NPDS est un outil puissant, dont la langue "maternelle" est le Français, mais qui sait aussi parler d'autres langues (l'Anglais est livré de base)

La section d'Administration est le centre de "commandement" de NPDS.

Depuis la version Narval, NPDS dispose d'une capacité Multi-langues. "ML" indique si cette capacité est active ou non (couleur rouge).
Les modules d'administration :
La présence des modules de ce centre de commandement est règlée par le profil de l'Administrateur connecté (voir le module Editer des administrateurs)

Les XX derniers articles :
Affiche les XX derniers articles publiés sur votre site. Vous devriez voir comme premier enregistrement, le dernier article publié.

Certains articles sont indiqués en 'archive', ce sont les articles ayant dépassés leur date de fin de publication.
Pour remettre ces articles dans le flux, il suffit de les éditer, de cliquer sur l'option 'changer la date' et de modifier leur date de fin de publication.

Cliquez sur l'article pour l'éditer ou utilisez les liens Editer, Effacer. Si vous voulez Editer/Effacer un ancien article, saisissez sont ID et sélectionnez l'action appropriée.
